SELEKTORY

Selektory CSS mogą się odnosić do elementów HTML, takich jak: konkretne znaczniki, identyfikatory (id), klasy (class) oraz inne atrybuty.

1. SELEKTOR PROSTY

Jeżeli slektorem jest dowolny znacznik języka HTML, to mamy do czynienia z selektorem prostym. Formatowanie znaczników polega na definiowaniu selektorów za pomocą ich nazw.

h2 { color: red; }

Można również grupować selektory znaczników, które mają mieć identyczne właściwości.

h1, p, i { color: red; }

2. SELEKTOR ID

Do identyfikacji poszczególnych elementów używa się identyfikatorów (w skrócie id). Każdy identyfikator ma unikalną nazwę w obszarze jednego dokumentu. Tę nazwę można wykorzystać jako selektor do formatowania elementu, który identyfikuje. Zapis selektora id zawsze poprzedzamy #.


Dla elementu <div id="stopka"></div> definiowanie formatowania w CSS bedzie wyglądać:

#stopka { text-align: center; color: green; }

3. SELEKTOR CLASS

Inną metodą definiowania selektora jest klasa. Klasy definiuje się atrybutem class. Selektor klasy odszukuje wiele elementów z tą samą nazwą klasy. Atrybut klasy można używać wielokrotnie w obrębie jednego dokumentu, dzięki czemu jesteśmy w stanie formatować wiele elementów jednocześnie. Zapis selektora class zawsze poprzedzamy ".".


Dla elementu <div class="stopka"></div> definiowanie formatowania w CSS bedzie wyglądać:

.stopka { text-align: center; color: red; }

Jeśli chcemy formatować część takich samych znaczników, również możemy użyć atrybutu class.

Dla elementu <p class="inny"> Tekst </p> definiowanie formatowania w CSS bedzie wyglądać:

p.inny { color: red; }

SELEKTORY

W CSS stosuje się również selektory uniwersalne, oznaczone znakiem *. Są one najwyższego poziomu tak jak body.


Hierarchiczny układ elementów HTML pozwala na stosowanie selektorów zależnych od elementów umieszczonych wyżej w hierarchii, na tym samym poziomie lub niżej. Mówimy wtedy o selektorach potomka, selektorach dziecka lub selektora braci.


W skrypcie wyżej wykorzystano selektor potomka. Tylko element li który jest potomkiem ul został zmieniony.
