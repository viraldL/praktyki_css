DEFINICJA CSS

CSS - (kaskadowe arkusze stylów ang. Cascading Style Sheets) jest językiem przeznaczonym do tworzenia arkuszy (plików), za których pomocą określa się cechy poszczególnych znaczników, czyli po prostu wygląd strony.

STRUKTURA ARKUSZA STYLÓW

Style CSS składają się z listy reguł (dyrektyw).

Każda reguła składa się z selektora i deklaracji.

Deklaracja składa się z właściwości, którym przypisuje się określone wartości.

Skłądnia pojedyńczej reguły wygląda w następujący sposób:

selektor { właściwość1: wartość1; właściwość2: wartość2; właściwość3: wartość3; }

STRUKTURA ARKUSZA STYLÓW

Selektorem może być wybrany element strony lub grupa elementów (elementy można grupować za pomocą atrybutów id i class). Wewnątrz nawiasów klamrowych deklaruje się cechy danego elementu w postaci właściwości. Każdej właściwości po dwukropku przyporządkowuje się wartości. Zapis każdej deklaracji kończymy średnikiem.

Przykładowa deklaracja dla każdego elementu p na stronie:

p { text-align: center; color: blue; }

(wszystkie paragrafy zostaną wyśrodkowane i zapisane niebieską czcionką)

METODY DOŁĄCZANIA STYLÓW

Istnieją trzy metody dołączania kaskadowych arkuszy stylów do dokumentów HTML.


  1. Metoda inline (styl wpisany, lokalny). W tej metodzie stosuje się atrybut style dla wybranego znacznika jezyka HTML.
  2. Metoda osadzenia w nagłówku <head> (styl wewnętrzny). Taki arkusz stylów definiuje się w nagłówku dokumentu HTML, wewnątrz znaczników <style> oraz </style>.
  3. Metoda łączenia z plikiem zewnętrznym (styl zewnętrzny). Polega na utworzeniu oddzielnego pliku, który zawiera listę reguł. Plik zapisujemy rozszerzeniem .css, a następnie definiujemy polączenie z nim za pomocą znacznika <link> w nagłowku dokumentu HTML następującym kodem: <link rel="stylesheet" type="text/css" href="nazwa.css">

Oprócz tego każda z przeglądarek ma dodatkowo własne wbudowane arkusze stylów.
