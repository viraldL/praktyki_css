MODEL BLOKOWY

Model blokowy w CSS opisuje bloki, które są tworzone dla elementów HTML. Definiuje ustawienie marginesów: zewnętrznych i wewnętrznych, obramowania i zawartości każdego elementu.


Struktura modelu blokowego:

MODEL BLOKOWY

margin - margines zewnętrzny


padding - margines wewnętrzny


border - obramowanie

Przykładowy blok z tekstem. (width: 200px;
height: 200px;)
