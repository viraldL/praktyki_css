KASKADOWOŚĆ

Kaskadowość arkuszy stylów określa kolejność oddziaływania stylu zależnie od metody dołączenia kodu do formatowanego dokumentu HTML

KASKADOWOŚĆ

Style o wyższym priorytecie mają pierwszeństwo w modyfikowaniu elementów dokumentu HTML. Style o niższym priorytecie mogą zmieniać formatowanie jakiegoś elementu tylko wtedy jeżeli styl o wyższym priorytecie nie definiują formatowania tego elementu.


KOMENTARZE W CSS

Tak samo jak w dokumentach HTML, w CSS możemy dodawac komentarze za pomocą składni:

/* komentarz do arkuszu CSS */

DZIEDZICZENIE

Struktura każdego dokumentu HTML opiera się na hierarchicznej budowie drzewa dokumentu. Każdy element w drzewie dokumentu ma dokładnie jednego rodzica, oprócz elementu głównego, czyli korzenia drzewa, którym jest zawsze element <HTML>.


DZIEDZICZENIE

Pozycja w danej hierarchi decyduje również o dziedziczeniu pewnych cech (np. wyglądu) od elementów znajdujących się wyżej. Nie wszystkie właściwości CSS są dziedziczone automatycznie. Niektóre elementy HTML mają domyślne właściwości (np. h4).



(element p oraz div odziedziczyły od elementu body właściwość font-size. Element h3 nie odziedziczył właściwości, ponieważ ma swój nadrzędny styl.)

DZIEDZICZENIE

Właściwości, które nie są dziedziczone:


  • background,
  • border,
  • margin,
  • padding,
  • width,
  • height,
